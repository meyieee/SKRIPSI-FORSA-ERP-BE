# controlfile-scm-be
Create the backend server of SCM &amp; control file modules via Node.js, Express.js &amp; MySQL

## Code Rules & Guidelines
1. Controller name format: METHOD - MODULE NAME. Example: `getCompany` or `updateReadingList`
2. API endpoint name format:
- GET all ('/customers')
- GET by ID ('/customer/:id')
- POST ('/customer')
- PUT ('/customer/:id')
- DELETE ('/customer/:id')
